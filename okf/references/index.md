# References

- **DXF — Drawing Interchange Format** — AutoCAD's ASCII drawing format; SwiftDXF reads the
  `ENTITIES` (model-space) section.
- **`ezdxf`** (MIT) — the reference reader SwiftDXF is validated bit-for-bit against:
  <https://ezdxf.mozman.at/>
